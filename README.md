# Projet Platine

**Description courte :**
Qcess est une plateforme SaaS modulaire qui permet de gérer intelligemment les bâtiments (résidences, entreprises...) avec un système d'accès par QR code et des modules activables selon les besoins de chaque organisation.

## Objectif du projet

Le but du projet est de créer une solution complète de gestion de bâtiments offrant :
- Une gestion centralisée des accès via QR codes sécurisés
- Un système modulaire adaptable aux besoins spécifiques (résidence, entreprise)
- Une interface d'administration web pour les gestionnaires
- Une application mobile pour les utilisateurs finaux
- La possibilité d'activer/désactiver des modules selon les besoins

---

## Architecture

L'application est construite selon une **architecture hexagonale** (Ports & Adapters) permettant une séparation claire entre la logique métier et les détails techniques.

Le système de modules pluggables permet d'ajouter de nouvelles fonctionnalités sans modifier le code existant.
